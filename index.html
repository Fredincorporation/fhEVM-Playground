Node.js & npm

Why: Node.js is the JavaScript runtime that powers Hardhat, the development framework for Ethereum smart contracts.

  1. Visit nodejs.org and download the LTS version (v20+)
  2. Run the installer and follow the setup wizard
  3. Verify installation by opening PowerShell and typing:
    node --version && npm --version

‚úì npm is included with Node.js automatically

2

Git

Why: Git is needed to clone the fhEVM Playground repository and manage version control.

  1. Download Git from git-scm.com
  2. Run the installer with default settings (or customize as needed)
  3. Verify installation by opening PowerShell and typing:
    git --version

‚úì Use Git Bash for a better terminal experience on Windows

3

Docker (Optional)

Why: Docker isolates your environment, preventing conflicts with other projects and ensuring consistency.

  1. Download Docker Desktop for Windows
  2. Install and enable WSL 2 backend (recommended)
  3. Verify installation:
    docker --version

Xcode Command Line Tools

Why: Required for compiling native modules and using git on macOS.

  1. Open Terminal and run:
    xcode-select --install
  2. Follow the prompts to install the tools (takes ~5 minutes)
  3. Verify installation:
    xcode-select -p
2

Node.js & npm (via Homebrew)

Why: Node.js powers the development environment and Hardhat framework for smart contracts.

  1. Install Homebrew if not already installed:
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Install Node.js:
    brew install node
  3. Verify:
    node --version && npm --version
3

Git

Why: Essential for cloning repositories and version control.

  1. Install via Homebrew:
    brew install git
  2. Configure your identity:
    git config --global user.name "Your Name"
    git config --global user.email "your@email.com"
  3. Verify:
    git --version

Node.js & npm

Why: Node.js runtime for Hardhat and fhEVM development environment.

  1. Ubuntu/Debian:
    curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
    sudo apt-get install -y nodejs
  2. Fedora/RHEL:
    sudo dnf install nodejs npm
  3. Verify:
    node --version && npm --version

Git

Why: Clone repositories and manage version control.

  1. Ubuntu/Debian:
    sudo apt-get install -y git
  2. Fedora/RHEL:
    sudo dnf install -y git
  3. Configure:
    git config --global user.name "Your Name"
    git config --global user.email "your@email.com"

Build Tools

Why: Required for compiling native modules used by some npm packages.

  1. Ubuntu/Debian:
    sudo apt-get install -y build-essential python3
  2. Fedora/RHEL:
    sudo dnf groupinstall "Development Tools" && sudo dnf install python3
